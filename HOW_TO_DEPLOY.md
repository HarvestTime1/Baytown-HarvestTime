# How to Deploy Updates to the HTCB App

## Every time you get a new file from Claude:

### Step 1
Download the file Claude gives you (e.g. `htcb_final_v4.html`)

### Step 2
Go to:
👉 **github.com/HarvestTime1/Baytown-HarvestTime**

### Step 3
Click on `index.html` in the file list

### Step 4
Click the **pencil icon ✏️** (top right of the file view)

### Step 5
Press **Ctrl+A** to select everything → press **Delete**

### Step 6
Open the file Claude gave you → **Ctrl+A** → **Ctrl+C** to copy it

### Step 7
Click back in GitHub → **Ctrl+V** to paste

### Step 8
Scroll down → click **"Commit changes"** → click **"Commit changes"** again

### Done ✓
Vercel picks it up automatically and deploys in about 30 seconds.

---

## That's it. No Codespaces. No terminal. No git commands.
## GitHub web editor → Vercel auto-deploys. Every time.
